Gemini Session Configuration

To use the free chatbot, you need to authenticate with Gemini Web using your browser's session cookies.

How to get cookies:
  1. Go to gemini.google.com and sign in.
  2. Press F12 to open Developer Tools.
  3. Go to Application -> Storage -> Cookies -> https://gemini.google.com.
  4. Find and copy the values for __Secure-1PSID and __Secure-1PSIDTS.
